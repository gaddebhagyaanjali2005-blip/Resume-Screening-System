# 📄 Resume Screening System

## 🚀 Overview

Resume Screening System is a Machine Learning and NLP-based application that automatically classifies resumes into different job categories. It uses text preprocessing, TF-IDF vectorization, and a LinearSVC model to predict the most suitable category for a resume.

## ✨ Features

* Resume text classification
* NLP-based text preprocessing
* TF-IDF feature extraction
* Machine Learning prediction
* Interactive Streamlit web application

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* NLP (Natural Language Processing)
* Streamlit
* Git & GitHub

## 📊 Dataset Information

* Total Resumes: 2484
* Categories: 24

## 🤖 Machine Learning Model

* Algorithm: LinearSVC
* Feature Extraction: TF-IDF
* Accuracy: 68%

### Install Dependencies

```bash

```

### Train the Model

```bash
python train.py
```

### Run the Application

```bash
streamlit run app.py
```
